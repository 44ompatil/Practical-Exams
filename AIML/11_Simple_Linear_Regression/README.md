# Simple Linear Regression

## What is it?
Imagine plotting dots on a graph showing Height vs Weight. Taller people tend to weigh more, so the dots generally slope upwards. 

If you take a ruler, place it on the paper, and map a straight line that slices perfectly through the center of all those dots—you just did **Linear Regression**.

By drawing that line, you can now predict the future! If someone tells you a height that *isn't* on the graph, you just trace your finger up to the ruler's line to easily guess their weight.

## Why "Simple"?
It is "Simple" because it only looks at **ONE** input to guess the output. 
Example: Guessing Salary based *only* on Years of Experience.

## How to run this code?
Make sure `scikit-learn`, `numpy`, and `matplotlib` are installed.
Run `python main.py` in your terminal! It will save an image showing a line cutting through data dots.
