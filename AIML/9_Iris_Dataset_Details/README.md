# The Iris Dataset

## What is it?
If you take a beginner coding class, the first thing you type is "Hello World." 
If you take an Artificial Intelligence class, the first dataset you touch is the **Iris Dataset**. It is the "Hello World" of Data Science!

## Where did it come from?
In 1936, a statistician walked into a garden and measured the petals and leaves of 150 different Iris flowers. 
There were 3 different species of Iris flowers in the garden. For each flower, he wrote down:
1. The length of the petal
2. The width of the petal
3. The length of the sepal (leaf)
4. The width of the sepal

Because this data is so clean and perfect, it has been used for 90 years to train beginner AI models to look at measurements and guess what species of flower it is!

## How to run this code?
Make sure `scikit-learn` and `pandas` are installed.
Run `python main.py` in your terminal to view the raw data!
