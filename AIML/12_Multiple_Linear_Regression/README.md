# Multiple Linear Regression

## What is it?
This is the same as Simple Linear Regression, but much more realistic. 
Instead of looking at just ONE thing to make a prediction, it looks at **MULTIPLE** things.

## Example
If you are trying to predict the price of a House, you can't *just* look at the square footage. You also need to look at:
1. Number of Bedrooms
2. Age of the house
3. Number of bathrooms

Multiple Linear Regression takes all these different inputs, figures out how important each one is (e.g. giving Bedrooms a higher weight than Age), and calculates a final predicted price!

## How to run this code?
Make sure `scikit-learn` and `numpy` are installed.
Run `python main.py` in your terminal!
