# K-Means Algorithm

## What is it?
Imagine you walk into a cafeteria filled with 100 strangers. You are told to divide them into 3 groups based on who they are sitting near. 
You would probably look around, find 3 general "clumps" of people, and draw an imaginary circle around each clump.

This is exactly what **K-Means Clustering** does. 
- The **K** stands for the number of groups you want (e.g., K=3 means you want 3 groups).
- **Means** refers to the center point (the average) of each group.

## Why use it?
It is an "Unsupervised" learning model. This means you don't give the AI any answers. You just hand it raw data, like a giant list of customer ages and incomes, and say: "Please find 3 natural groupings in this data so we can run targeted marketing campaigns." The AI finds the clumps for you!

## How to run this code?
Make sure `scikit-learn` and `matplotlib` are installed.
Run `python main.py` in your terminal! It will create an image showing groups of colored dots!
