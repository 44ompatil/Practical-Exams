# Practical 17: Layout Widgets (Row, Column, Container)

## Organizing the Screen
Think of building a screen like organizing your desk. 

* **Container**: A box that you can color, resize, and add padding to. It usually holds exactly one item (like a pencil holder holding a pencil).
* **Row**: Arranges items horizontally from left to right (like laying out your pens side by side on the desk).
* **Column**: Arranges items vertically from top to bottom (like stacking your books).

By blending these, we can build absolutely any UI we want!
