# Practical 18: Design with Row and Column

## Creating Grids
In this practical, we deeply combine `Row` and `Column` to create a grid-like structure. 
A column organizes multiple rows vertically. Each of those rows organizes multiple containers horizontally. 

This is the very foundation of how massive apps like Instagram or WhatsApp are built on your screen. The screen is essentially just a giant invisible spreadsheet of Rows and Columns!
