# Practical 1: Variables and Data Types

Welcome to the very first step in programming! 

## What is a Variable?
Imagine you are moving to a new house, and you use cardboard boxes to pack your things. You write "Books" on one box and "Clothes" on another. In programming, a **Variable** is just like that cardboard box. It is an empty space where you can store information, and you stick a label (a name) on it so you can find it later.

## What is a Data Type?
If you have different kinds of items, you might use different types of containers. You wouldn't put hot soup in a cardboard box; you'd use a thermos! Similarly, the computer needs to know what *kind* of data you are trying to store, so it can allocate the right type of virtual box.

Here are the main sizes and shapes of our virtual boxes used in the code:
* **String**: A box meant for text. Example: `"Alice"` (Notice we use quotes so the computer knows it's text).
* **int**: Short for integer. A box meant for whole numbers. Example: `22`
* **double**: A box meant for numbers with decimal points. Example: `5.9`
* **bool**: Short for boolean. A tiny box that can only hold a simple `true` (yes) or `false` (no). Example: `true`.

By telling the computer the exact type of box we need, we make our programs run efficiently without breaking down!
