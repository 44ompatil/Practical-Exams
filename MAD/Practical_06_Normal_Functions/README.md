# Practical 6: Normal Functions

## What is a Function?
A function is like a cooking recipe. Imagine you bake a cake every weekend. Instead of figuring out the exact ingredients and steps every single time, you write the recipe down on a card and name it "My Chocolate Cake." Now, whenever you want cake, you just pull out that card and follow it.

In programming, a function bundles a chunk of code together so we can reuse it easily just by calling its name.

There are two main types of functions we show here:

1. **Functions that just DO things**: 
   Our `greetUser()` function just prints out a welcome message. It doesn't give us anything back; it just performs an action. (We use `void` to show it's returning "nothing").

2. **Functions that GIVE back an answer**:
   Our `calculateSum(a, b)` function takes in two numbers, does the math, and **returns** (gives back) the final answer. We tell the program to expect an `int` (a whole number) back from it. 

Functions keep our code clean, organized, and reusable!
