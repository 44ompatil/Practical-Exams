# Practical 5: Loops

## Doing Things Over and Over
Imagine having to write "I will not talk in class" 100 times on a whiteboard. It would take ages! A computer never gets tired, so we give it a **Loop**, which is an instruction to repeat a specific piece of code multiple times.

### 1. The `For` Loop
This loop is used when you know *exactly* how many times you want to repeat something. It has a built-in counter. 
*"Count from 1 to 3, and each time, print the number."*

### 2. The `While` Loop
This loop acts more like an ongoing condition. 
*"**While** it is still raining, keep your umbrella open."* 
It checks the condition first. If it's true, it runs the code. As soon as the condition becomes false, it stops.

### 3. The `Do-While` Loop
This is almost identical to the While loop, but with a twist: It guarantees the action will happen **at least once** before checking the condition.
*"Eat a slice of pizza. Then, assess if you are still hungry."* Even if you weren't that hungry, you ate the first slice before checking!
