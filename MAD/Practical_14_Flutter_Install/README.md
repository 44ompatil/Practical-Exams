# Practical 14: Install Flutter and Run a Basic App

## Step 1: Install Flutter
1. Go to the official Flutter website (flutter.dev/docs/get-started/install).
2. Download the installation bundle for your operating system (Windows, Mac, or Linux).
3. Extract the downloaded zip file into a location like `C:\src\flutter`.
4. Add the `flutter\bin` folder to your System Environment Path variable so your computer knows where Flutter is from any folder in terminal.
5. Run `flutter doctor` in your command prompt to see if you are missing anything like Android Studio.

## Step 2: Create a Basic App
1. Open your terminal or command prompt.
2. Type `flutter create my_first_app` and hit Enter. The computer will generate all the necessary files automatically!
3. Go into the folder by typing `cd my_first_app`.

## Step 3: Run the App
1. Make sure you have a device connected or an emulator running.
2. Type `flutter run` in the terminal to launch your app.
